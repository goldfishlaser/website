LED Fashion



LED umbrella is a project where LED strands are attached to a transparent umbrella to add cheer to rainy days. The LED strands are controlled by a Gemma microcontroller.

RepRap Prusa

RepRap 3D printer, Prusa varient was a project to build a 3D printer that I did with assistance from Ben Morgan. A notable thing about this build is that we improved the Prusa design by incorporating linear bearings in place of some of the usual printed parts of the Y axis to create a smoother print experience. The printer soon clogged, and work was halted on this project, but not before creating a cool glitchy Pink Panther Lady. Although it wasn't long running, I learned a lot about the machine from the process of building it.

ThreeJS Music Box

Music Box Music box is a demo I wrote to practice the threejs javascript library. The site contains a music box that can be wound up so that it opens and plays a song. This demo is in progress. It currently does not work very well on mobile and it takes some time to load. On desktop, use your cursor to click on the latch to open the music box. Next, click and drag to turn the music box so that you can see the turnkey on its back. Click on the turnkey to turn it, and when you have turned it sufficiently, the music will begin to play. Click the latch again to stop the music and close the box.

OpenSCAD 3D Models

Midcity coaster [STL, Github] Midcity coaster is a project to create some swag from scratch! I used Inkscape to create a DXF file from the Midcity logo. I used OpenScad to create an STL. The STL was put into Flashpoint to create gcode so that the result could be printed on my Flashforge Creator Pro.

Alchemy symbol [SCAD, Github] Alchemy symbol is a project to model a simple geometric shape using OpenScad.
