Designing Parts for print

Objects printed are designed on the computer using CAD programs. There are many open source tools available including:

  • OpenSCAD
  • FreeCAD
  • Heekscad
  • Blender
  • Google Sketchup
  • Art of Illusion

 

You can also choose from proprietary software such as the following:

  • PTC Creo
  • Solidworks
  • Autodesk Inventor
  • 3D Studio Max
  • Alias

 

Which software you use is a matter of your personal preference. Many RepRap users choose Google Sketchup, Blender, or OpenSCAD, and some use several different tools depending on what they are designing.

 

With these programs you will primarily be interested in two types of files, CSG files and STL files. CSG files will ensure that you can edit your objects in any of the programs above. When you are ready to print, you will convert your CSG file to an STL file so that your CAM tool can interpret it for use by the RepRap machine.

 

CAM tools include:

  • Skeinforge
  • RepSnapper
  • RepRap Host Software
